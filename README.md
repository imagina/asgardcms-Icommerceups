# asgardcms-icommerceups

## Seeder

    run php artisan module:seed Icommerceups

## Vendors

    - add composer.json 
        "gabrielbull/ups-api": "^0.7.6"

    - That's all :)
    

## Configurations

    - Access Key
    - User ID
    - Password
    - Mode (Sandbox or Live)
    - Shipper Postalcode
    - Shipper Statecode
    - Shipper Countrycode

## API

    https://icommerce.imagina.com.co/api/icommerceups
